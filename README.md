# 📚 OOP Project - Java

## 📌 Overview
This project is based on Object-Oriented Programming (OOP) concepts implemented in Java.  
It demonstrates core principles like Encapsulation, Inheritance, and Abstraction using different classes.

# 📁 File Structure

```
OOP-Project/
│
├── README.md
├── UML.pdf
│
└── src/
    ├── person.java
    ├── admin.java
    ├── bookable.java
```

## 🧱 Project Structure
The project includes the following main classes:

- `person.java` → Base class representing a person  
- `admin.java` → Derived class for admin functionality  
- `bookable.java` → Interface or class representing bookable entities  
- `UML.pdf` → UML diagram showing system design  

## 🎯 OOP Concepts Used
- Encapsulation  
- Inheritance  
- Abstraction  
- Class & Object usage  

## ⚙️ How to Run

1. Open terminal / command prompt  
2. Navigate to project folder  
3. Compile files:

```bash
javac *.java
```

4. Run the main class:

```bash
java Main   // (if you have a main class, otherwise specify correct class)
```
